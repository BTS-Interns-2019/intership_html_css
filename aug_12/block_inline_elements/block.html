Block Elements

Edgar David Peregrino Jimenez

12/02/2019

Un elemento en bloque ocupa todo el espacio de su elemento padre (contenedor), creando así un "bloque".

A continuacion se enumeran los elementos de bloque usados por HTML:

<address>

Información de contacto

<article>

Contenido adicional

<aside>

Contenido adicional

<audio>

Reproductor de audio

<blockquote>

Bloque de cita

<canvas>

Dibujo

<dd>

Descripcion de definición

<div>

División de documento

<dl>

Lista de definición

<fieldset>

Etiqueta de conjunto de campos

<figcaption>

Leyenda de figira

<figure>

Contenido multimedia con una leyenda, ver figcaption

<footer>

Sección de pagina

<form>

Formulario de entrada

<h1>,<h2>,<h3>,<h4>,<h5> y <h6>

Niveles de cabecera (1 -6)

<header>

Sección o cabecera de pagina

<hgroup>

Grupos información de encabezado

<hr>

Regla horizontal (linea de división)

<li>

Elemento de lista

<main>

Engloba el único contenido central del documento

<nav>

Contiene enlaces de navegaciópn

<noscript>

Contenido para ser usado si los scripts no son soportados o permitidos

<ol>

Lista ordenada

<output>

Formulario de salida

<p>

Parrafo

<pre>

Texto preformateado

<section>

Sección de una pagina web

<table>

Tabla

<tfoot>

Pie de tabla

<ul>

Lista no ordenada

<video>

Reproductor de video
