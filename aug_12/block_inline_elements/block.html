Block elements

Address

The <address> tag in HTML indicates a field that contains contact information.

<address></address>

Article

The <article> tag in HTML indicates an article, that means a self-contained composition in a document, page, application or site.

<article></article>

Aside

The <aside> tag in HTML indicates a sidebar of the document, it is a portion indirectly related with the main content.

<aside></aside>

BlockQuote

The <blockquote> tag in HTML indicates a block for extended quotations.

<blockquote></blockquote>

Details

The <details> tag in HTML creates a disclosure widget in which information is visible only when the widget is toggled into an "open" state. It contains a tag named <summary> which specifies the summary, caption or legend for details.

<details></details>

Dialog

The <dialog> tag in HTML shows a dialog box or other interactive component, such as inspector or window.

<dialog></dialog>

DD

The <dd> tag in HTML indicates the description of a <dt> in a <dl> list.

<dd></dd>

Div

The <div> tag in HTML is the generic container for flow content. It doesn't have effects until it was styled in css.

<div></div>

DL

The <dl> tag in HTML indicates a description list.

<dl></dl>

DT

The <dt> tag in HTML indicates the term to be described in a <dl> list.

<dt></dt>

FieldSet

The <fieldset> tag in HTML indicates a field for contain the forms. It contains a tag named <legend> which is the caption of the fieldset.

<fieldset></fieldset>

FigCaption

The <figcaption> tag in HTML indicates the legend or caption that describes a <figure> element.

<figcaption></figcaption>

Figure

The <figure> tag in HTML contains an image or a group of images.

<figure></figure>

Footer

The <footer> tag in HTML indicates the footer which contains the author information, copyright data or links to related documents.

<footer></footer>

Form

The <form> tag in HTML is a section that contains interactive controls for submitting information to a web server. The method can be get or post.

<form></form>

H1

The <h1> tag in HTML indicates the first level of section headings.

<h1></h1>

H2

The <h2> tag in HTML indicates the second level of section headings.

<h2></h2>

H3

The <h3> tag in HTML indicates the third level of section headings.

<h3></h3>

H4

The <h4> tag in HTML indicates the fourth level of section headings.

<h4></h4>

H5

The <h5> tag in HTML indicates the fifth level of section headings.

<h5></h5>

H6

The <h6> tag in HTML indicates the sixth level of section headings.

<h6></h6>

Header

The <header> tag in HTML indicates a group of introductory or navigational aids.

<header></header>

HGroup

The <hgroup> tag in HTML indicates a group of headers.

<hgroup></hgroup>

HR

The <hr> tag in HTML indicates a horizontal rule or line, represents a thematic break between paragraph-level elements.

<hr>

Li

The <li> tag in HTML represents an item in a list.

<li></li>

Main

The <main> tag in HTML indicates the dominant content of the <body> of a document.

<main></main>

Nav

The <nav> tag in HTML indicates a section of navigation links.

<nav></nav>

OL

The <ol> tag in HTML represents a ordered list of items, typically rendered as numbered list.

<ol></ol>

P

The <p> tag in HTML indicates a paragraph.

<p></p>

Pre

The <pre> tag in HTML is a container with text which is presented exactly as written in the HTML file.

<pre></pre>

Section

The <section> tag in HTML represents a standalone section, is used for styling purposes.

<section></section>

Table

The <table> tag in HTML represents a two dimensional table, comprised of rows and columns of cells containing data.

<table></table>

UL

The <ul> tag in HTML represents a unordered list of items, typically rendered as bulleted list.

<ul></ul>
