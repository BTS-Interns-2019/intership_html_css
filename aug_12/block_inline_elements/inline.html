Inline elements

    <a>

    a element (or anchor element) along with his href attribute, creates a hyperlink to other web pages, files, locations, within the same page, or any URL.

    <abbr>

    abbr element represents an abbreviation or acronym; the optional title attribute can provide an expansion or description for the abbreviation

    <b>

    This was formerly known as the Boldface element. It's just another way to create a boldface text. Instead of this use the CSS font-weight property.

    <bdi>

    Bidirectional Isolate element (bdi) order the text in a specific direction using the Unicode Bidirectional Algorithm

    <bdo>

    Bidirectional Text Override element (bdo) overrides the current directionality of text, so that the text within is rendered in a different direction.

    <br>

    br element produces a line break in text (carriage-return). It is useful for writing a poem or an address, where the division of lines is significant.

    <button>

    button element represents a clickable button, which can be used in forms or anywhere in a document.

    <canvas>

    canvas element is used to draw graphics and animations with either the canvas scripting API or the WebGL API

    <cite>

    Citation element (cite) is used to describe a reference to a cited creative work, and must include the title of that work. The reference may be in an abbreviated form according to context-appropriate conventions related to citation metadata

    <code>

    code element displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code

    <data>

    data element links a given content with a machine-readable translation.

    <datalist>

    datalist element contains a set of option elements that represent the values available for other controls.

    <del>

    del element represents a range of text that has been deleted from a document. This can be used when rendering "track changes" or source code diff information, for example.

    <dfn>

    Definition Element (dfn) is used to indicate the term being defined within the context of a definition phrase or sentence.

    <em>

    em element marks text that has stress emphasis. The em element can be nested, with each level of nesting indicating a greater degree of emphasis.

    <embed>

    embed element embeds external content at the specified point in the document. This content is provided by an external application or other source of interactive content such as a browser plug-in.

    <i>

    i element element for text that is set off from the normal prose for readability reasons. This would be a range of text with different semantic meaning than the surrounding text.

    <iframe>

    Inline Frame element (iframe) embeds external content at the specified point in the document. This content is provided by an external application or other source of interactive content such as a browser plug-in.

    <img>

    img element embeds an image into the document.

    <imput>

    input element is used to create interactive controls for web-based forms in order to accept data from the user.

    <ins>

    ins element represents a range of text that has been added to a document. You can use the del element to similarly represent a range of text that has been deleted from the document.

    <kbd>

    Keyboard Input element (kbd) represents a span of inline text denoting textual user input from a keyboard, voice input, or any other text entry device

    <label>

    label element represents a caption for an item in a user interface.

    <map>

    map element is used with area elements to define an image map

    <mark>

    Mark Text element represents text which is marked or highlighted for reference or notation purposes, due to the marked passage's relevance or importance in the enclosing context.

    <meter>

    meter element represents either a scalar value within a known range or a fractional value.

    <noscript>

    noscript element defines a section of HTML to be inserted if a script type on the page is unsupported or if scripting is currently turned off in the browser.

    <object>

    object element represents an external resource, which can be treated as an image, a nested browsing context, or a resource to be handled by a plugin.

    <output>

    out is a container element into which a site or app can inject the results of a calculation or the outcome of a user action.

    <picture>

    picture tagcontains zero or more <source> elements and one img element to provide versions of an image for different display/device scenarios. The browser will consider each child source element and choose the best match among them; if no matches are found, the URL of the img element's src attribute is selected. The selected image is then presented in the space occupied by the img element.

    <progress>

    progress element displays an indicator showing the completion progress of a task, typically displayed as a progress bar.

    <q>

    q element indicates that the enclosed text is a short inline quotation. Most modern browsers implement this by surrounding the text in quotation marks.

    <ruby>

    ruby element represents a ruby annotation. Ruby annotations are for showing pronunciation of East Asian characters.

    <s>

    s element renders text with a strikethrough, or a line through it. Use the s element to represent things that are no longer relevant or no longer accurate.

    <samp>

    samp element is used to enclose inline text which represents sample (or quoted) output from a computer program.

    <script>

    script element is used to embed or reference executable code; this is typically used to embed or refer to JavaScript code.

    <select>

    select element represents a control that provides a menu of options

    <slot>

    slot element is a placeholder inside a web component that you can fill with your own markup, which lets you create separate DOM trees and present them together.

    <small>

    small element represents side-comments and small print, like copyright and legal text, independent of its styled presentation.

    <span>

    span element is a generic inline container for phrasing content, which does not inherently represent anything.

    <strong>

    strong element indicates that its contents have strong importance, seriousness, or urgency. Browsers typically render the contents in bold type.

    <sup>

    Subscript element specifies inline text which should be displayed as subscript for solely typographical reasons. Subscripts are typically rendered with a lowered baseline using smaller text.

    <template>

    Content template (template) element is a mechanism for holding HTML that is not to be rendered immediately when a page is loaded but may be instantiated subsequently during runtime using JavaScript.

    <textarea>

    textarea element represents a multi-line plain-text editing control, useful when you want to allow users to enter a sizeable amount of free-form text, for example a comment on a review or feedback form.

    <time>

    time element represents a specific period in time. It may include the datetime attribute to translate dates into machine-readable format, allowing for better search engine results or custom features such as reminders.

    <u>

    str element represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation. This is rendered by default as a simple solid underline, but may be altered using CSS.

    <var>

    Variable element (var) represents the name of a variable in a mathematical expression or a programming context.

    <video>

    video element embeds a media player which supports video playback into the document. You can use video for audio content as well, but the audio element may provide a more appropriate user experience.
