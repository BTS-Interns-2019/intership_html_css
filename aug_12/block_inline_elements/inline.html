Inline elements

A

The <a> tag in HTML using href attribute, create a hyperlink to other web pages.

<a></a>

Abrr

The <abbr> tag in HTML indicates an abbreviation or acronym, using title attribute, it allows to display a description of the abbreviation.

<abbr></abbr>

Audio

The <audio> tag in HTML indicates an audio which can play into the document. It may contain one or more audio sources, represented using the src attribute or the <source>.

<audio></audio>

B

The <b> tag in HTML indicates a bold font type.

<b></b>

BdI

The <bdi> tag in HTML is text that may contain both sequences of characters that are arranged left-to-right (LTR) and sequences of characters that are arranged right-to-left (RTL), such as an Arabic quotation embedded in an English string.

<bdi></bdi>

BdO

The <bdo> tag in HTML overrides the current directionality of text, so that the text within is rendered in a different direction. Used in languages when the direction of test is different.

<bdo></bdo>

Big

The <big> tag in HTML increases one level the size of the text. It is considered obsolete.

<big></big>

Br

The <br> tag in HTML indicates a line break in text.

<br>

Button

The <button> tag in HTML indicates a clickable button.

<button></button>

Canvas

The <canvas> tag in HTML indicates a canvas scripting API or WebGL API to draw graphics and animations.

<canvas></canvas>

Cite

The <cite> tag in HTML is used to describe a reference to a cited creative work, and must include the title of that work. It employs with <a> elements to show references styled.

<cite></cite>

Code

The <code> tag in HTML indicates a text styled in computer code

<code></code>

Data

The <data> tag in HTML links a given content with a machine-readable translation.

<data></data>

Datalist

The <datalist> tag in HTML indicates a container of <option> elements.

<datalist></datalist>

Del

The <del> tag in HTML indicates a range of text that has been deleted from a document.

<del></del>

Dfn

The <dfn> tag in HTML is used to indicate the term being defined within the context of a definition phrase or sentence.

Em

The <em> tag in HTML marks text that has stress emphasis. It is represented as italic font type.

<em></em>

Embed

The <embed> tag in HTML embeds external content into the document.

<embed>

I

The <i> tag in HTML represents text that is set off from the normal prose. It is represented as italic font type.

<i></i>

IFrame

The <iframe> tag in HTML represents a nested browsing context, embedding another HTML page into the current one.

<iframe></iframe>

Img

The <img> tag in HTML indicates an image into the document.

<img/>

Input

The <input> tag in HTML is used for input values in a form.

<input>

Ins

The <ins> tag in HTML indicates a range of text that has been added to a document.

<ins></ins>

Kbd

The <kbd> tag in HTML is a span of inline text denoting textual user input from a keyboard, voice input, or any other text entry device.

<kbd></kbd>

Label

The <label> tag in HTML indicates a caption for an item in a user interface.

<label></label>

Map

The <map> tag in HTML is used with <area> elements to define an image map.

Mark

Meter

NoScript

The <noscript> tag in HTML indicates a part of HTML that will execute when the Script is not supported.

<noscript></noscript>

Object

The <object> tag in HTML represents an external resource, which can be treated as an image, a nested browsing context, or a resource to be handled by a plugin.

<object></object>

Output

The <output> tag in HTML is a container element into which a site or app can inject the results of a calculation or the outcome of a user action.

<output></output>

Picture

The <picture> tag in HTML contains zero or more <source> elements an one <img> element to provide versions of an image for different display/device scenarios.

<picture></picture>

Progress

The <progress> tag in HTML displays an indicator showing the completion progress of a task, typically displayed as a progress bar.

<progress></progress>

<q></q>

Script

<script></script>

Select

The <select> tag in HTML represents a control combobox that provides a menu of <option> elements.

<select></select>

Slot

The <slot> tag in HTML is a placeholder inside a web component that you can fill with your own markup, which lets you create separate DOM trees and present them together.

<slot></slot>

Small

The <small> tag in HTML represents side-comments and small print, like copyright and legal text, independent of its styled presentation.

<small></small>

Span
