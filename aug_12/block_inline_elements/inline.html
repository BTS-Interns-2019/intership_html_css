Inline Elements

A Tag

The HTML a element (or anchor element), along with its href attribute, creates a hyperlink to other web pages, files, locations within the same page, email addresses, or any other URL.

Usado para representar links

Abbr Tag

The HTML Abbreviation element (abbr) represents an abbreviation or acronym; the optional title attribute can provide an expansion or description for the abbreviation. If present, title must contain this full description and nothing else.

Usado para las abreviaciones si pasas el cursor sobre el te dara su definicion.

Audio Tag

The HTML audio element is used to embed sound content in documents. It may contain one or more audio sources, represented using the src attribute or the source element: the browser will choose the most suitable one.

Usado para colocar audios en la pagina.

B Tag

The HTML Bring Attention To element (b) is used to draw the reader's attention to the element's contents, which are not otherwise granted special importance. This was formerly known as the Boldface element, and most browsers still draw the text in boldface. However, you should not use b for styling text; instead, you should use the CSS font-weight property to create boldface text, or the strong element to indicate that text is of special importance.

Bold Text

BDI Tag

The HTML Bidirectional Isolate element (bdi) tells the browser's bidirectional algorithm to treat the text it contains in isolation from its surrounding text. It's particularly useful when a website dynamically inserts some text and doesn't know the directionality of the text being inserted.

Usado para contener diferentes tipos de escritura.

BDO Tag

The HTML Bidirectional Text Override element (bdo) overrides the current directionality of text, so that the text within is rendered in a different direction.

Usado igualmente para diferentes tipos de escritura.

BR Tag

The HTML br element produces a line break in text (carriage-return). It is useful for writing a poem or an address, where the division of lines is significant.

Es un salto de línea.

Button Tag

The HTML button element represents a clickable button, which can be used in forms or anywhere in a document that needs simple, standard button functionality.

Es un botón.

Canvas Tag

Use the HTML canvas element with either the canvas scripting API or the WebGL API to draw graphics and animations.

Espacio para dibujar.

Cite Tag

The HTML Citation element (cite) is used to describe a reference to a cited creative work, and must include the title of that work. The reference may be in an abbreviated form according to context-appropriate conventions related to citation metadata.

Citas Bibliográficas.

Code Tag

The HTML code element displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code. By default, the content text is displayed using the user agent's default monospace font.

Usado para marcar algun fragmento de codigo en un documento.

Data Tag

The HTML data element links a given content with a machine-readable translation. If the content is time- or date-related, the time element must be used.

Usado para insertar datos ocultos que se verán al pasar por encima del elemento.

Datalist Tag

The HTML datalist element contains a set of option elements that represent the values available for other controls.

Listas de selección de datos

Del Tag

The HTML del element represents a range of text that has been deleted from a document. This can be used when rendering "track changes" or source code diff information.

usado para marcar texto como eliminado (rayado y en rojo)

Dfn Tag

The HTML Definition element (dfn) is used to indicate the term being defined within the context of a definition phrase or sentence. The p element, the dt/dd pairing, or the section element which is the nearest ancestor of the dfn is considered to be the definition of the term.

Em Tag

The HTML em element marks text that has stress emphasis. The em element can be nested, with each level of nesting indicating a greater degree of emphasis.

Italicas

Embed Tag

The HTML embed element embeds external content at the specified point in the document. This content is provided by an external application or other source of interactive content such as a browser plug-in.

Sirve para incrustar cualquier contenido multimedia especificado en el type.

I Tag

The HTML i element represents a range of text that is set off from the normal text for some reason. Some examples include technical terms, foreign language phrases, or fictional character thoughts. It is typically displayed in italic type.

Italic Text

Iframe Tag

The HTML Inline Frame element (iframe) represents a nested browsing context, embedding another HTML page into the current one.

Contiene el contenido de otra pagina indicada en sus propiedades.

Img Tag

The HTML img element embeds an image into the document.

Usado para contener imagenes.

Input Tag

The HTML input element is used to create interactive controls for web-based forms in order to accept data from the user; a wide variety of types of input data and control widgets are available, depending on the device and user agent.

Define un input text.

Ins Tag

The HTML ins element represents a range of text that has been added to a document.

Marca el texto como agregado (color verde).

Kbd Tag

The HTML Keyboard Input element (kbd) represents a span of inline text denoting textual user input from a keyboard, voice input, or any other text entry device. By convention, the user agent defaults to rendering the contents of a kbd element using its default monospace font, although this is not mandated by the HTML standard.

Representa la entrada de teclado.

Label Tag

The HTML label element represents a caption for an item in a user interface.

Es solo una etiqueta

Map Tag

The HTML map element is used with area elements to define an image map (a clickable link area).

Usado para insertar enlaces en cierta area del documento html, con map declaramos el mapeo de la zona y con area denotamos el area para el enlace.

Mark Tag

The HTML Mark Text element ( mark ) represents text which is marked or highlighted for reference or notation purposes, due to the marked passage's relevance or importance in the enclosing context.

Usado para remarcar texto.

Meter Tag

The HTML meter element represents either a scalar value within a known range or a fractional value.

Es una barra indicadora parecida a progreso pero te mide el valor de un estado con relacion a los parametros que se le dictan.

Noscript Tag

The HTML noscript element defines a section of HTML to be inserted if a script type on the page is unsupported or if scripting is currently turned off in the browser.

Object Tag

The HTML object element represents an external resource, which can be treated as an image, a nested browsing context, or a resource to be handled by a plugin.

Es un objeto versatil a usarse como se desee.

Output Tag

The HTML Output element ( output ) is a container element into which a site or app can inject the results of a calculation or the outcome of a user action.

Utilizado para mandar una salida a otro sitio o app.

Picture Tag

The HTML picture element contains zero or more source elements and one img element to provide versions of an image for different display/device scenarios.

Usado para incluir una o mas fotos en un contenedor HTML las distribuira de forma correcta.

Progress Tag

The HTML progress element displays an indicator showing the completion progress of a task, typically displayed as a progress bar.

Basicamente una barra de progreso

Q Tag

The HTML q element indicates that the enclosed text is a short inline quotation. Most modern browsers implement this by surrounding the text in quotation marks.

Ruby Tag

The HTML ruby element represents a ruby annotation. Ruby annotations are for showing pronunciation of East Asian characters.

S Tag

The HTML s element renders text with a strikethrough, or a line through it. Use the s element to represent things that are no longer relevant or no longer accurate.

Basicamente es un texto rayado.

Samp Tag

The HTML Sample Element ( samp ) is used to enclose inline text which represents sample (or quoted) output from a computer program.

Texto en un formato especial

Script Tag

The HTML script element is used to embed or reference executable code; is typically used to embed or refer to JavaScript code.

Select Tag

The HTML select element represents a control that provides a menu of options.

Slot Tag

The HTML slot element is a placeholder inside a web component that you can fill with your own markup, which lets you create separate DOM trees and present them together.

Small Tag

The HTML small element represents side-comments and small print, like copyright and legal text, independent of its styled presentation. By default, it renders text within it one font-size small, such as from small to x-small.

Insertar letras pequeñas (contratos)

Span Tag

The HTML span element is a generic inline container for phrasing content, which does not inherently represent anything.

Strong Tag

The HTML Strong Importance Element ( strong ) indicates that its contents have strong importance, seriousness, or urgency. Browsers typically render the contents in bold type.

Negritas

Sub Tag

The HTML Subscript element ( sub ) specifies inline text which should be displayed as subscript for solely typographical reasons. Subscripts are typically rendered with a lowered baseline using smaller text.

Subindices

Sup Tag

The HTML Superscript element ( sup ) specifies inline text which is to be displayed as superscript for solely typographical reasons. Superscripts are usually rendered with a raised baseline using smaller text.

Superindices

Template Tag

The HTML Content Template ( template ) element is a mechanism for holding HTML that is not to be rendered immediately when a page is loaded but may be instantiated subsequently during runtime using JavaScript.

Textarea Tag

The HTML textarea element represents a multi-line plain-text editing control, useful when you want to allow users to enter a sizeable amount of free-form text, for example a comment on a review or feedback form.

Area de Texto para comentarios.

Time Tag

The HTML element represents a specific period in time. It may include the datetime attribute to translate dates into machine-readable format, allowing for better search engine results or custom features such as reminders.

U Tag

The HTML Unarticulated Annotation Element ( u ) represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation.

Subrayado.

Var Tag

The HTML Variable element ( var ) represents the name of a variable in a mathematical expression or a programming context. It's typically presented using an italicized version of the current typeface, although that behavior is browser-dependent.

Representación de variables.

Video Tag

The HTML Video element ( video ) embeds a media player which supports video playback into the document.

Contenedor para videos.

Wbr Tag

The HTML wbr element represents a word break opportunity—a position within text where the browser may optionally break a line, though its line-breaking rules would not otherwise create a break at that location.
