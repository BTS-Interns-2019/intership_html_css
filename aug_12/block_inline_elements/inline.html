Elementos en linea

Por Ricardo Martin del Campo Loza

12/08/2019

b

b de bold = negrita
Indica que el texto debe ser representado con una variable bold de la tipografía que se esté usando.

big

big de big = grande
Muestra el texto marcado con un tamaño de funete ma sgrande. El tamaño de la funte puede definirse en una escala del uno al 7, big aumenta este valor en un punto.

i

Muestra el texto marcado con un estilo en cursiva o italica

small

Hace el tamaño del texto una talla más pequeña que el tamaño minimo de fuente del navegador.

tt

Muestra el texto marcado con una fuente de ancho fijo

abbr

Representa una abreviación o acrónimo; el atributo opcional title puede ampliar o describir la abreviatura.Si está presente, el atributo title debe contener la descripción completa y nada mas

acronym

acronym = acrónimo
Es el encargado para marcar formas abreviadas.Además, gracias al atributo title podemos indicar la versión extendida del término.

cite

cite = cita
Marca una referencia a una fuente, o el autor de un texto citado.

code

Es el apropiado par amarcar el código de un programa.

dfn

de define = definir
Sirve para marcar el término que se quiere definir.

em

De emphasis = énfasis
Es el apropiado para marcar con énfasis las partes importantes de un texto.

kbd

Marca el texto que debe introducir el usuario.

strong

El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto.

samp

Señala lo que es una salida de un programa.

time

Puede incluir el atributo datetime para convertir las fechas en un formato interno legible por un ordenador.
Puede representar uno de los contenidos siguientes
- Una hora en formato de 24 horas.
- Una fecha en el calendario gregoriano.

var

Marca variaables de programas y similares

a

Crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

bdo

Bi-Directional Overriding=Anulación de bidireccionalidad.
Permite un control total de la direccionalidad de escritura de texto.

br

Line break produce un salto de linea en el texto .

img

Representa una imagen en el documento.

map

Es usado para definir una imagen/mapa del lado del cliente(nua imagen con areas clicables).

object

Define un objeto embebido multimedia, como audio, video, applets, PDF, etc.

q

Crea citas en linea, marca las citas a otros autores o documentos.

script

Se utiliza para insertar o hacer referencua a un script ejecutable dentro de un documento HTML o XHTML

span

span = abarcar
Sirve para aplicar estilo al texto o agrupar elementos en línea.

sub

Define un fragmento de quexto que se debe mostrar, por razones tipográficas, más bajo; y generalmente más pequeño, es decir, en subíndice.

sup

Define un fragmento de texto que se debe mostrar, por razónes tipográficas, más alto y generalmente más pequeño, es decir, en superíndice

button

Representa un elemento cliqueable de tipo botón que puede ser utilizado en formularios o en cualquier parte de la página que necesite un botón estándar y simple de aplicar.

input

Se usa para crear controles interactivos para formularios basados en la web, que reciban datos del usuario.

label

Representa una etiqueta para un elemento en una interfaz de usuario.

select

Representa un control que muestra un menú de opciones.

textarea

Representa un control para edición muti-línea de texto plano.
