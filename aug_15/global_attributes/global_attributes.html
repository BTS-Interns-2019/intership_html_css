Atributos Globales

Los atributos globales son atributos comunes a todos los elementos HTML; pueden usarse en todos los elementos, aunque pueden no tener efecto en algunos elementos.

ATRIBUTO DESCRIPCION
ACCESSKEY Proporciona un atajo de tavlado para el elemento actual.
AUTOCAPITALIZE Controla si la entrada de texto se cambia a mayusculas o no, puede tener los siguientes valores:
off o none: se aplica la no auto modificacion del texto.
on o sentences: la primera letra de cada enuncado se convierte en mayusculas.
words: La primera letra de cada palabra se hace mayuscula y las demas en misnusculas.
characteres: todas las letras por default pasan a mayusculas.
CLASS Una lista separada por espacios en las clases del elemento.
las classes que en css y javascript seleccionen y acedan a elementos especificos.
atravez de desectores especificos o funciones como el letodo:
Document.getElementByClassName().
CONTENTEDITABLE Atributo que indica si el elemento debe ser editable por el usuario.
Si es asi el davegador modifica su widget para permitir la edicion.
el atributo debe tomar uno de los sifuientes valores:
  • True: que indica que el elementod debe ser editable.
  • False: lo que indica que elemento no debe der editable.
CONTEXTMENU La identificacion de un < Menú> para usar como emnu contextual para este elemento
DATA-* Forma una clase de atributos, llamados atributos datos personalizados, que permiten intercambiar informacion propietaria entre el HTMLy su representacion DOM que pueden ser utilizados por los scripts
DIR Un atributo enumerado que indica la direccionalidad del texto del elemento. Puede tener los siguientes valores:
  • ltr: Lo que significa de izquierda a derecha y se debe utilizar para los idiomas que se escriben de izquierda a derecha (como el inglés);
  • rtl : Lo que significa de derecha a izquierda y debe usarse para idiomas que se escriben de derecha a izquierda (como el árabe);
  • auto: Permite que el usuario decida.
DRAGGABLE Unatributo enumerado que indica si el elemento suede arrastrar, utilizando la API de arrastrar y solter. puede tener los siguientes valores:
  • True: pudede ser arrastrado
  • False: no pueden ser arrastrado
DROPZONE Un atributo enumerad que indica que ripos de contenido se pueden soltar en este elemento utilizando la API de arrastrar y soltar. Pueden tener los siguientes valores:
  • copy: Que indica que soltar creara una cipia del elemento.
  • move: que indica que el elemento que se arrastro se movera a esta nueva ubucacion
  • link: creara un enlce a los datos arrastrados
exportparts Se usa para exportar de forma transtiva partes de sombra ne un arbol de sombra anidado.
hideen Un atributo boleano indica que el elemento aun no es, o ya no es relevante.
ID Define un identificador unico que debe ser unico en todo el docmuento
inputmode Proporciona una posta a los navegadores sobre el tipo de configuracion de teclado vitual que se ultilizara al editar este elemento o su contenido.
is Le permite especificar que un elemento HTML estandar debe comportarse como un elemento incorporado personalizado registrado.
itemif El Identificador unico y global en un item
itemprop Se usa para agregar propiedades a un elemento. Cada elemento HTML puede tener un atributo itemprop especificado.
itemref LAS propuedades que nos son decendientes de un elemento con el atributo itemscope se pueden asociar al elemento mediante una referencia de elemento.
itemscope Funciona junto con itetype para especificar que el HTML contenido en un bloque es sobre un elemento particular.
itemtype Especifica la URL del voacbulario que se ultiliza para definir iremprops en la estructura de datos.
lang Ayuda a definir el idioma de un elemento.
part una lista separada por espacios de los nombres de las partes del elemento.
slot Asigna una ranura de un shadow DOM shadow: un elemento con un atributo se asigna a la ranura creada por ele elemento < slot>
spellcheck Un atributo enumerado define si se puede veridicar el el elemnto tiene errores otrograficos.
style contiene declaraciones de estilo CSS.
tabindex Unatributo entero que indica si el elemento puede tomar el foco de entrada:
  • a:negative significa que el elemento debe ser enfocable
  • 0: significa que el elemento debe ser enfocable y accesible atravez de la navegacion secuencial.
  • a:positive significa que no puede ser enfocable
title Contiene un texto que representa informacion de asesoramiento relacionada con el elemento al que pertenece
translate Un atributo enumerado que se utiliza para especificar los valors del atributo de un elemento y los valores de sus elementos secundarios de nodo de texto se traduciran cuando se localice la paguina o si se dejaran sin cambios.
