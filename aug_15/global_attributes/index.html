Atributos globales

Los atributos globales son aplicables a todos los elementos HTML, algunos no tienen efecto en ciertos elementos. Cual quier atibuto no estandar aun permite estos atributos. Los atributos como su nombre lo dice comandos o propiedades de una tag que permite dar modificaciones especificas para que tome diceño forma y funcionalidad en web.

Acces key
Atajo de teclado, aunque cada navegador tienen su forma de entrar al atajo definido, para chrome, edge, explorer y safari, opera es alt + key y para firefox es alt+shift+key
autocapitalize
Atributo enumerado que controla si la entrada de texto se capitaliza automaticamente. Este atributo debe te tener uno de estos valores:
  • off o none: no se aplica, todas las letras son minusculas.
  • on o sentences la primera letra tiene como valor predeterminado mayuscula.
  • words: La primera letra de cada palabra por defecto es una letra mayuscula, todas las demas predeterminan minusculas.
  • characters: todas las letras deben estar en mayusculas.
class
Identificador que dan acceso a CSS y Javascript al elemento con este atributo
contentenditable
Indica si el elemento debe ser editable por el usuario. true false
contextmenu
El idde a menu para usar como menú contextual para este elemento.
data-*
Forma una clase de atributos llamados atributos de datos personalizados, que permiten intercambiar informacion propietaria entre HTML y su representacion DOM que pueden ser utilizados por los scripts disponible con HTMLElemnt.
dir
Atributo de texto que indica direccionalidad de texto.
  • ltr: de izquierda a derecha
  • rtl: de derecha a izquierda
  • auto: permite que el agente de usuario decida
Draggable
Atributo enumerable que indica si el elemento se puede arrastrar utilizando apis de arrastrar y soltar, puede tener true o false
dropzone
Atributo enumerable que indica que tipos de contenido se pueden soltar en un elemento, utilizando apis de arrastrar y soltar.
  • copy: creara una copia
  • move: indica que lo arrastrado se movera a una nueva ubicacion
  • link: creará un elnlace a los datos arrastrados.
exportparts
Se utiliza para exportar de forma transitiva partes sombreadas de un arbol sombreado anidado a un arbol ligero que lo contiene.
hidden
Atributo booleano que indica si un leemento es o no relevanto. Lo oculta
id
define un identificador unico que no se debe repetir en el documento HTML. Permite seleccionar ese elemento desde CSS o Javascript
inputmode
espesifica al navegador que teclado virtual se utilizará al editar el elemento.
is
Le permite especificar que un elemento HTML estándar debe comportarse como un elemento incorporado personalizado registrado
itemid
El identificador único y global de un artículo.
itemprop
Se usa para agregar propiedades a un elemento. Cada elemento HTML puede tener un itempropatributo especificado, donde un itempropconsiste en un par de nombre y valor.
itemref
Las propiedades que no son descendientes de un elemento con el itemscopeatributo pueden asociarse con el elemento mediante un itemref. Proporciona una lista de identificadores de elementos (no itemids) con propiedades adicionales en otras partes del documento.
itemscope
itemscope(generalmente) funciona junto con itemtypepara especificar que el HTML contenido en un bloque es sobre un elemento en particular. itemscopecrea el artículo y define el alcance de lo itemtypeasociado con él.
itemtype
Especifica la URL del vocabulario que se utilizará para definir itemprops (propiedades del elemento) en la estructura de datos. itemscopese usa para establecer el alcance de dónde en la estructura de datos el vocabulario establecido por itemtypeestará activo.
lang
Ayuda a definir el idioma de un elemento, en el que esta o en el que debe escribir
part
Una lista separada por espacios de los nombres de las partes del elemento. Los nombres de partes permite CSS
slot
Asigna un espacio a un arbol de sombre DOM.
spellcheck
Un atributo enumerado define si se puede verificar si el elemento tiene errores ortográficos. true o false
style
Contiene estilo css inline
tabindex
Atriibuto que indica si el elemento puede tomas el foco de entrada, si deberia participar en la navegacion secuencial del teclado.
  • un valor negativo significa que el elemento debe ser enfocable, pero no debe ser accesible a través de la navegación secuencial del teclado
  • 0 significa que debe ser enfacable
  • Un valor positivo signidica que el elemento debe ser enfocable y accesible a traves de la navegacion del teclado
tle
Contiene un texto representa la informacion de asesoramiento relacionada con el elemento al que pertenece.
translate
Un atributo enumerado que se utiliza para especificar si los valores de atributo de un elemento y los valores de sus Texthijos de nodo se traducirán cuando se localice la página, o si se dejarán sin cambios. Yes o No
