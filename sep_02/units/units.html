CSS Units

The CSS Units are used to set lengths, paddings, margins, align elements and so on.

There are two types of Units used in CSS: Absolute and Relative.


Absolute Units

The Absolute Units are not relative to anything else and are generally considered to always be the same size.

Unit Name Equivalent to
cm Centimeters 1cm = 96px/2.54
mm Millimeters 1mm = 1/10th of 1cm
Q Quarter-millimeters 1Q = 1/4th of 1mm
in Inches 1in = 2.54cm = 96px
pc Picas 1pc = 1/16th of 1in
pt Points 1pt = 1/72th of 1in
px Pixels 1px = 1/96th of 1in

Relative Units

The Relative Units are relative to something else, perhaps the size of the parent element's font, or the size of the viewport.

The benefit of using relative units is that with some careful planning you can make it so the size of text or other elements scale relative to everything else on the page.

Unit Relative to
em Font size of the parent element.
ex x-height of the element's font.
rem Font size of the root element.
lh Line height of the element.
vw 1% of the viewport's width.
vh 1% of the viewport's height.
vmin 1% of the viewport's smaller dimension.
vmax 1% of the viewport's larger dimension.
