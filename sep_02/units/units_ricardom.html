Css values and units

Every property in CSS ahs a value or set of values that are allowed for that property.
In CSS there are various usefull numeric data types, the following are all classed as numeric.

Numbers, lengths , and percentages

Integer

An integer is a whole number such as 1024 or -55.

Number

A number represents a decimal number --- it may or may not have a decimal point with a fractional component. For example 0.255, 128 - 1.2.

Dimension

A dimension is a number with a unit atached to it, for example 45deg, 5s, or 10px. dimension is an umbrella category that includes the length, angle, time and resolution types.

Percentage

A percentage represents a fraction of some other value, for example 50%. Percentage is allways relative to another quantity, for example an element's length is relative to its parent element's length.

Lengths

There are two types of length used in CSS -- relative and absolute.

Absolute length units

Th efollowing are all absolute length units -- they are not relative to anything alse and are generally considered to allways be the same size

Unit Name Equivalent to
cm Centimeters 1cm = 96px/2.54
mm Millimeters 1mm = 1/10th of 1cm
Q Quarter-millimeters 1Q = 1/40th of 1cm
in Inches 1in = 2.54cm = 96px
pc Picas 1pc = 1/16th of 1in
pt Points 1pt = 1/72th of 1in
px Pixels 1px = 1/96th of 1in

this values are more commonly used for print, but for screen output its not very used in exception of px(pixels).

Relative length units

relative length units are relative to something else, perhaps the size of the parent element's font, or the size of the viewport. The benefit of using relative units is that with some careful planning you can make it so the size of text or other elements scale relative to everything else on the page.

Unit Relative to
em Font size of the parent element.
ex x-height of the element's font.
rem Font size of the root element.
lh Line height of the element.
vw 1% of the viewport's width.
vh 1% of the viewport's height.
vmin 1% of the viewport's smaller dimension.
vmax 1% of the viewport's larger dimension.

Percentages

In a lot of cases a percentage is treated in the same way as a length. the thing with percentages is that they are always set relative to some other values.

Numbers

Some values accept numbers, withput any unit added to them.

Color

There are many ways to specify color in CSS, some of whitch are more recently implemented than others. The same color values can be used everywere in CSS, wether you are specifying text color, background color, or whatecer else

Color keywords

Colo keywords are a seimple and understable way of specifying color.

Hexadecimal RGB values

The next type of color value you are likely to encounter is hexadecimal codes. Each hex value consists of a hash/pound symbol (#) followed by six hexadecimal numbers, each of which can take one of 16 values between 0 and f

RGB and RGBA values

The third scheme we'll talk about here is RGB.An RGB value is a function which is given three parameters that represent the red, green, and blue channel values of the colors, in much the same way as hex values.Each chanel is represented bu a decimal number between 0 and 255

HSL and HSLA values

Slightly less well-supported then RGB is the HSL color model, which was implemented after much interest from designers.Instead of red, green and blues , function accepts hue, saturarion, and lightness.

Images

The image data type is used is used wherever an image is a valid value.

Position

The position data type represents a set of 2D coordinates, used to position an item such as background image. It can take a keyword such as top, left, bottom, right and center.
